# Moraberdampak  
Template Website Pesantren Berbasis CodeIgniter 4

**Moraberdampak** adalah template website pesantren berbasis **CodeIgniter 4 (CI4)** yang dirancang untuk mendukung **transformasi digital pesantren**.  
Project ini bersifat **reusable**, ringan, aman, mudah dikonfigurasi, dan **SEO-ready** sehingga dapat digunakan oleh banyak pesantren.
